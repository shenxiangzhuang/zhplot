## 快速开始

### 安装

使用 pip 安装`zhplot`:

```bash
pip install zhplot
```

使用 uv 安装`zhplot`:

```bash
uv add zhplot
```

### 使用方法

使用`zhplot`非常简单，只需在脚本开头导入即可：
```diff
+ import zhplot
import matplotlib.pyplot as plt
```

### 一个简单的例子
<div align="center">
    <img src="https://github.com/shenxiangzhuang/zhplot/blob/7569552e07a8b4de7afd8c9df5cbcb154a349e97/docs/images/zhplot_demo.png?raw=true" width="500"/>
</div>

```python
import zhplot
import matplotlib.pyplot as plt


plt.plot([1, 2, 3, 4])
plt.title('这是一个标题')
plt.xlabel('横坐标')
plt.ylabel('纵坐标')
plt.show()
```
